# 🌿 Greenden - Plant Store Website

Greenden is a responsive plant store website built using **HTML** and **Tailwind CSS**. The project focuses on creating a clean, modern UI with proper layout techniques like **Flexbox** and **Grid**, ensuring a smooth user experience across devices.

## 🚀 Features

* Responsive design for mobile, tablet, and desktop
* Modern UI using Tailwind CSS utility classes
* Navigation bar with mobile hamburger menu
* Hero section with call-to-action buttons
* Product listing page with grid layout
* Search bar UI for product filtering
* About section with feature highlights
* Customer review section
* Newsletter subscription form
* Footer with branding

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS
* Basic JavaScript (for mobile menu interaction)

## 📱 Responsiveness

The website is fully responsive and adapts using:

* Flexbox for layout alignment
* Grid system for product display
* Media queries via Tailwind (`md`, `lg` breakpoints)

## 📂 Project Structure

* `index.html` – Home page
* `products.html` – Product listing page
* `images/` – Contains all assets used in the project

## 🎯 Purpose

This project was developed to practice and demonstrate:

* UI/UX design skills
* Responsive web design
* Tailwind CSS styling
* Layout structuring using Flexbox and Grid
